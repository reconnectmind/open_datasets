verified = проверено по первоисточнику, а не выведено из описания. Пустая ячейка означает «не подтверждено», а не «отсутствует».

Запись

Минимальная сырая единица, которую можно анализировать независимо: для BIDS NIRS — уникальный participant × session × task × run файл, для NIRx — один stem файлов, для WFDB — одна запись. Это не число участников или trials; sidecars и events.tsv счётчик не увеличивают.

Длина fNIRS

Min–max длительности одной независимо анализируемой raw fNIRS-записи, а не сумма по датасету. verified вычислен по samples/time vector/header, partial основан на протоколе или неполной границе, unavailable означает, что диапазон обосновать нельзя.

Источник

Репозиторий/платформа, где опубликован датасет (OpenNeuro, Zenodo, OSF, Mendeley, Dryad, …), вместе с версией записи. Каталоги и репозитории сами по себе в таблицу не включены.

Доп. признаки

Сопутствующие данные помимо fNIRS: поведение, демография, клинические, опросники, физиология, движение, оптоды/пространство, стимулы, другие модальности. Перечислены только положительно подтверждённые.
